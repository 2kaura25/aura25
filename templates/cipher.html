CIPHER-CHASE



A digital treasure hunt where teams solve puzzles, crack codes, and uncover hidden clues.

Rules and Regulations

1. Each team must have 2 members.

2. Each team gets one problem file, two program files, and an individual laptop.

3. Unlock password-protected files and run the correct program in Python.

4.Complete the challenge within the given time.

5.First team to unlock files and execute the correct program wins.

6.No mobile phones allowed.

7.No password sharing.

8.Any rule violation leads to immediate disqualification.
